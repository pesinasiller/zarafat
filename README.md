



  # ظرافت

  ## نصب و اجرا

  ۱. وابستگی‌ها را نصب کنید:
    ```
    npm install
    ```

  ۲. اگر Sanity CLI را نصب نکرده‌اید، به صورت سراسری نصب کنید:
    ```
    npm install -g @sanity/cli
    ```

  ۳. استودیو Sanity را اجرا کنید:
    ```
    npm run studio
    ```

  ۴. مرورگر خود را باز کنید و به [http://localhost:3333](http://localhost:3333) بروید.
